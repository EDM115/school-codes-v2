Description détaillée

Rectangle de couleur affiché dans un panneau. Les rectangles ont comme caractéristiques :

  • une couleur
  • une position
  • une taille
  • une vitesse (un déplacement élémentaire par unité de temps)

Un rectangle est immobile par défaut (même si une vitesse est indiquée). Ce sera à vous d'en faire un objet mobile et autonome dans la classe dérivée "Mobile".
