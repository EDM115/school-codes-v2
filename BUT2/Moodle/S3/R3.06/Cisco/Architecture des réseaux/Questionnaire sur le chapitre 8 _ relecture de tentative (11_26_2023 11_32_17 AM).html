Texte de la question

Pourquoi attribue-t-on une adresse IP à l'interface VLAN1 sur un commutateur de couche 2 Cisco ?
Veuillez choisir une réponse.
Pour créer un nouveau réseau local IP sur le commutateur
Pour permettre le transfert de paquets IP par le commutateur
Pour autoriser l'accès à distance au commutateur afin de le gérer
Pour permettre au commutateur d'acheminer les paquets entre des réseaux

Feedback

Refer to curriculum topic: 8.5.1
Le VLAN 1 sur un commutateur de couche 2 Cisco est une interface virtuelle. Par défaut, le VLAN 1 est attribué comme interface de gestion pour l'accès à distance (Telnet ou SSH) à l'appareil à des fins de gestion. Afin de permettre l'accès à distance via le réseau, la configuration des adresses IP doit être correcte sur le VLAN 1.
La réponse correcte est : Pour autoriser l'accès à distance au commutateur afin de le gérer

Texte de la question

Un administrateur réseau travaille sur un routeur Cisco. L'invite CLI est Router1(config-if) #. Que va probablement configurer l'administrateur ensuite ?
Veuillez choisir une réponse.
Lignes vty
Interface LAN
Port de console
Mot de passe requis pour accéder au mode d'exécution privilégié

Feedback

Refer to curriculum topic: 8.3.1
Cisco IOS utilise différentes invites pour identifier les modes de configuration spécifiques. L'invite Router1(config-if)# est un mode de sous-configuration de Router(config)#. Ce mode indique que les commandes de configuration suivantes sont appliquées à une interface LAN ou WAN.
La réponse correcte est : Interface LAN

Texte de la question

Quels sont les deux appareils réseau utilisés dans les réseaux d'entreprise pour assurer la connectivité réseau aux terminaux ? (Choisissez deux réponses.)
Veuillez choisir au moins une réponse.
routeur
pare-feu
commutateur de réseau local
serveur web
point d'accès sans fil

Feedback

Refer to curriculum topic: 8.1.1
Les points d'accès sans fil et les commutateurs LAN sont déployés sur les réseaux d'entreprise pour permettre aux terminaux de se connecter sans fil au réseau ou via des câbles. Un routeur est utilisé pour permettre aux terminaux sur un réseau local de communiquer avec d'autres terminaux sur des réseaux distants. Un pare-feu protège le réseau en filtrant le trafic entrant et sortant. Un serveur web héberge le contenu web.
Les réponses correctes sont : commutateur de réseau local, point d'accès sans fil

Texte de la question

Quel sont les deux protocoles à utiliser pour accéder à un commutateur Cisco pour l'administration intraband ? (Choisissez deux réponses.)
Veuillez choisir au moins une réponse.
DHCP
SSH
FTP
SMTP
Telnet

Feedback

Refer to curriculum topic: 8.1.2
Telnet et SSH sont les deux protocoles à utiliser pour accéder à distance à un appareil réseau pour les tâches de gestion intrabande. DHCP fournit des informations de configuration IP dynamiques aux appareils clients. Le protocole FTP sert à transférer des fichiers entre un client et un serveur. Le protocole SMTP sert à livrer les messages électroniques.
Les réponses correctes sont : SSH, Telnet

Texte de la question

Quelle est la différence entre les termes « mot clé » et « argument » dans la structure des commandes IOS ?
Veuillez choisir une réponse.
Un mot clé est nécessaire pour effectuer une commande. Un argument ne l'est pas.
Un mot clé apparaît toujours directement après une commande. Un argument non.
Un mot clé est un paramètre spécifique. Un argument n'est pas une variable prédéfinie.
Un mot clé a une longueur prédéfinie. Un argument peut avoir n'importe quelle longueur.

Feedback

Refer to curriculum topic: 8.3.2
Les mots clés sont des paramètres spécifiques qui sont définis dans IOS. Les arguments ne sont pas prédéfinis, ils sont les variables définies par l'utilisateur pour exécuter une commande. Bien que la plupart des mots clés apparaissent directement après les commandes, ils peuvent apparaître à d'autres emplacements selon la structure des commandes. Un mot clé peut être saisi entièrement ou sous une forme raccourcie.
La réponse correcte est : Un mot clé est un paramètre spécifique. Un argument n'est pas une variable prédéfinie.

Note de 2,00 sur 2,00
Marquer la question

Texte de la question

Quel mode de Cisco IOS affiche une invite Router# ?
Veuillez choisir une réponse.
Mode de configuration globale
Mode d'exécution privilégié
Mode Configuration
Mode d'exécution utilisateur

Feedback

Refer to curriculum topic: 8.3.1
Cisco IOS utilise différents modes de commande pour les besoins opérationnels. Le mode d'exécution privilégié est identifié par une invite qui se termine par le symbole # après le nom de l'appareil. Le mode d'exécution utilisateur est identifié par une invite qui se termine par le symbole > après le nom de l'appareil. Le mode de configuration globale est identifié par une invite qui se termine par (config)# après le nom de l'appareil.
La réponse correcte est : Mode d'exécution privilégié

Pourquoi faut-il configurer une adresse de passerelle par défaut sur un hôte ?
Veuillez choisir une réponse.
Pour fournir une adresse permanente à un ordinateur
Pour identifier le réseau auquel un ordinateur est connecté
Pour identifier l'adresse logique d'un ordinateur sur le réseau
Pour identifier l'appareil qui permet aux ordinateurs du réseau local de communiquer avec des appareils sur d'autres réseaux
