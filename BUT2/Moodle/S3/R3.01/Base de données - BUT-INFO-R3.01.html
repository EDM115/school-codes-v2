    Base de données

    1. Travail à réaliser (2 séances)

    1. Définissez le diagramme de classes UML de la base de données utilisée par l’application Web SportTrack.

    2. Donnez le schéma relationnel résultant de la traduction du diagramme de classes UML précédent. Vous appliquerez les règles vues en cours l’année passée.

    3. Implantez le schéma relationnel obtenu dans une base de données SQLite3. Pour se faire, écrivez un script create_db.sql permettant de créer la base de données qui sera utilisée par votre application Web. Utilisez la commande sqlite3 pour créer la base de données SQLite3 à l’aide du script create_db.sql, et sauvegardez cette base dans le fichier sport_track.db.

    Vous pourrez vous référer à la documentation de SQLite3 à l’URL suivante: https://www.sqlite.org

    2. Rendu

    Vous devrez déposer en fin de séance sur Moodle une archive tar.gz contenant:

    • un fichier PDF présentant le diagramme de classes UML et le schéma relationnel de la base de données;
    • le script create_db.sql de création de la base de données.
