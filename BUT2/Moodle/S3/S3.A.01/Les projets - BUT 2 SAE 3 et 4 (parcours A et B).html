    Les projets

    Les S.A.É. des semestres 3 et 4 se feront à travers des projets, qui répondent à des besoins formulés par des clients. Les projets seront hébergés sur la forge logicielle Forgens, la forge GitLab mise à disposition par l’UBS.

    Ces projets étant réalisés pour des clients qui ont fait confiance au département Informatique de l’IUT de Vannes en leur proposant ces projets, ceux-ci devront être réalisés de manière rigoureuse et être de qualité. Ainsi, les rapports de spécifications devront être bien rédigés, être compréhensibles et être précis. Le code qui sera produit devra être de qualité, être performant, être parfaitement documenté et testé. Les commentaires qui seront laissés sur le dépôt Git du projet devront être explicites.

    Les projets proposés pour l’année 2023-2024 pour le parcours A sont les suivants:

    • LifeCompanion
    • Gestion d’un club d’escalade
    • Interface graphique QT pour l’utilisation d’un réseau de neurones de détection d’objets
    • Application Web pour l’utilisation d’un réseau de neurones de détection d’objets

    Les projets proposés pour l’année 2023-2024 pour le parcours B sont les suivants:

    • Cloud (NextCloud) reposant sur un Raspberry Pi sécurisé
    • NAS (OpenMediaVault) reposant sur un Raspberry Pi
